# SnailLift Cloudflare Pages

SnailLift Sprint 1A supports a Cloudflare Pages Wrangler-assisted path.

## Steps

1. Export Website ZIP.
2. Extract the ZIP into a folder such as `postsnail-public`.
3. Prepare SnailLift Cloudflare settings in the admin.
4. Copy the generated Wrangler command.
5. Run it locally with a limited Cloudflare Pages token.
6. Return to PostSnail and run Verify Live Site.
7. Notify Forest after verification passes.

## Token Guidance

Use a limited Cloudflare Pages token. Do not paste long-lived account-wide tokens unless you understand the risk.

Sprint 1A does not store tokens by default and does not claim browser direct upload is universally available.
